# Contributing to Prisma Shift

Thank you for your interest in contributing! This document provides guidelines for getting started.

## Development Environment

- **Node.js**: >= 18.0.0
- **npm**: >= 8.0.0
- **TypeScript**: 5.0+

## Setup

```bash
git clone https://github.com/eftech93/prisma-shift.git
cd prisma-shift
npm install
npm run build
npm test
```

## Branch Naming

- `feature/` — New features or enhancements
- `fix/` — Bug fixes
- `docs/` — Documentation improvements
- `chore/` — Maintenance tasks

## Pull Request Checklist

- [ ] Tests pass (`npm test`)
- [ ] Code compiles without errors (`npm run build`)
- [ ] New features include tests
- [ ] Documentation is updated (README, docs/)
- [ ] Commit messages follow [Conventional Commits](https://www.conventionalcommits.org/)

## Commit Message Format

```
type(scope): subject

body (optional)
```

Types: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`
