# Installation

## Prerequisites

- **Node.js** 18+ 
- **Prisma** 5.x
- **Database** supported by Prisma (PostgreSQL, MySQL, SQLite, SQL Server, MongoDB)

## Install via npm

```bash
npm install prisma-shift
```

Or with yarn:

```bash
yarn add prisma-shift
```

Or with pnpm:

```bash
pnpm add prisma-shift
```

## Verify Installation

```bash
npx prisma-shift --version
# 0.0.2
```
