# Release Notes v0.0.1

**Release Date:** March 24, 2024

---

## 🎉 Initial Release

Prisma Shift v0.0.1 is the first release of the TypeScript-based data migration tool for Prisma.

---

## ✨ Features

### Core Features

- **TypeScript-based data migrations** - Write migrations in TypeScript with full type safety
- **Prisma Client access** - Use the Prisma Client for complex data transformations
- **Migration tracking** - Track executed migrations in the database
- **Rollback support** - Rollback migrations with `down()` functions
- **CLI commands** - Full-featured CLI for managing migrations

### CLI Commands

| Command | Description |
|---------|-------------|
| `init` | Initialize data migrations directory |
| `create <name>` | Create a new migration |
| `run` | Run pending migrations |
| `status` | Show migration status |
| `rollback` | Rollback last migration |
| `reset` | Clear migration records |

### Prisma Integration

- **Generator support** - Integrate with Prisma's generator system
- **CLI wrapper** - `prisma-shift-migrate` drop-in replacement for Prisma CLI
- **Client extension** - `withDataMigrations()` for programmatic access

### Migration API

```typescript
import { DataMigration, MigrationContext } from "prisma-shift";

const migration: DataMigration = {
  id: "20240324120000_migration_name",
  name: "migration_name",
  createdAt: 1711281600000,
  
  async up({ prisma, log }: MigrationContext) {
    // Migration logic
  },
  
  async down({ prisma, log }: MigrationContext) {
    // Rollback logic
  },
};

export default migration;
```

## Installation

```bash
npm install prisma-shift
```

---

## Quick Start

```bash
# Initialize
npx prisma-shift init

# Create migration
npx prisma-shift create "add_user_preferences"

# Run migrations
npx prisma-shift run
```
